# Visual UX Designer

This skill ensures Sana looks and feels like a modern, high-quality application.

## Styling Principles
- **Modern Aesthetic**: Use consistent spacing, rounded corners, and subtle gradients.
- **Polish**: Ensure interactive elements have hover/active states and clear focus indicators.

## Responsive Design
- **Flexbox & Grid**: Leverage CSS Flexbox and Grid for layout to ensure responsiveness across mobile and desktop.
- **Media Queries**: Organize media queries within the relevant SCSS mixins or component styles.

## Interactive Feedback
- **Loading States**: Always show a loading indicator or skeleton screen during async operations.
- **Toast Notifications**: Use non-intrusive notifications for successes and errors.

## Visual Consistency
- **Sana Logo**: Always use the official logo from `assets/`.
- **Primary Colors**: Adhere to the defined palette in `style.scss`.
